> *"Most analysts read data. I make data speak — turning numbers into narratives, patterns into products, and insights into impact."*

I sit at the crossroads of **data science**, **machine learning**, and **software engineering** — a rare combination that most people keep in separate silos. Based in **Karachi, Pakistan**, shaped by the **University of Karachi (UBIT)**, and battle-tested through **competitive data analytics** — I don't just analyze data, I build intelligent systems around it.

From wrangling messy financial datasets to training YOLO models on live textile factory footage — my work is rooted in **real-world problems, not textbook exercises**.

<br/>

ML isn't a buzzword in my work — it's a practical tool for solving business problems.

**Models I've trained & deployed:**

- **Time Series Forecasting** — Prophet · ARIMA · XGBoost · CatBoost for demand & trend prediction
- **Classification** — Logistic Regression · Random Forest for churn prediction, loan acceptance
- **Computer Vision** — Custom **Ultralytics YOLO** trained on real factory data for fabric defect detection
- **Segmentation** — SAM (Segment Anything Model) for automated annotation pipelines
- **Unsupervised Learning** — K-Means Clustering · T-SNE · PCA for pattern discovery
- **RAG Systems** — LangChain-powered retrieval-augmented generation pipelines
- **Generative AI** — Gemini API integration for quiz generation, content automation

*Everything evaluated with proper metrics: PR curves, confusion matrices, ROC-AUC.*
